# Support

## Use The Right Channel

### Usage Question

Open a normal issue when you need help with:

- local development
- Astro build behavior
- locale smoke checks
- deployment expectations

### Bug Report

Open an issue with:

- expected behavior
- actual behavior
- route affected
- reproduction steps
- screenshots or logs if relevant

### Security Report

Do not open a public issue.

Use:

- `SECURITY.md`
