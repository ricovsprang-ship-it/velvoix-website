# Code Of Conduct

## Standard

This project expects respectful, direct, and professional collaboration.

Participants should:

- review in good faith
- keep feedback specific and technical
- avoid hostile or dismissive behavior
- respect private or sensitive information

## Unacceptable Behavior

- harassment
- personal attacks
- intimidation
- deliberate disclosure of sensitive information

## Scope

This applies to issues, pull requests, reviews, and related project
communication.

## Reporting

Report conduct problems privately to:

- `@ricovsprang-ship-it`
